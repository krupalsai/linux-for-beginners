# 🐧 Linux Basics

## What is Linux?

Linux is a free, open-source operating system kernel created by **Linus Torvalds** in 1991.
It powers everything from phones (Android) to servers to supercomputers.

## Popular Linux Distributions

| Distro | Best For |
|--------|----------|
| **Ubuntu** | Beginners |
| **Debian** | Stability |
| **Fedora** | Latest features |
| **Arch Linux** | Advanced users |
| **Kali Linux** | Cybersecurity |

## Linux File System Hierarchy

```
/               ← Root (everything starts here)
├── /home       ← User home directories
├── /etc        ← Configuration files
├── /var        ← Logs and variable data
├── /tmp        ← Temporary files
├── /bin        ← Essential binaries
├── /usr        ← User programs
└── /root       ← Root user's home
```

## The Terminal

The **terminal** (shell) lets you talk to Linux using text commands.
Default shell: **Bash** (Bourne Again SHell)

```bash
whoami          # shows your username
hostname        # shows computer name
uname -a        # shows OS info
```
