## Practice Environments

- **VirtualBox** — Free VM software to run Linux on Windows/Mac
- **WSL2** — Run Linux inside Windows without a VM
- **Replit / Codespaces** — Browser-based Linux terminal
- **KillerCoda** — Interactive Linux scenarios in browser

## YouTube Channels

- **NetworkChuck** — Fun Linux tutorials
- **TechHut** — Linux tips and distro reviews
- **LearnLinuxTV** — In-depth guides

## Recommended Path for Beginners

```
Week 1 → Linux Basics, file system, basic commands
Week 2 → File permissions, users, groups
Week 3 → Package management, scripting basics
Week 4 → Networking commands, cron jobs
Month 2 → Shell scripting, vim, SSH
```
