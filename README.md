## 🚀 Getting Started

1. Install a Linux distro (Ubuntu recommended for beginners)
2. Open a terminal
3. Start with [Linux-Basics.md](Linux-Basics.md)

> 💡 **Tip:** Practice every command in a safe VM or use [OverTheWire](https://overthewire.org) online.
